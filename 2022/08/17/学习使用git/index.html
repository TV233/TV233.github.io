使用 Git 将本地代码储存至 GitHub 仓库

# 第一步：创建 GitHub 仓库

点击 GitHub 官网右上角的加号，选新建仓库，输入仓库名，选择公共或私有，保存。

# 第二步：初始化一个本地仓库

找到本地代码的根路径，右键选择 Git Bash Here 。

执行命令 git init

（可以先在本地代码中新建一个 .gitignore 文件，这个文件中配置一些不需要 Git 托管的文件）

# 第三步：将本地工作目录中的文件添加到暂存区

执行命令 git add . （注意 add 后有空格和. ）

该命令将本地所有的文件添加到暂存区。

# 第四步：将暂存区中的代码提交到本地仓库

执行命令 git commit -m "填写本次提交的描述"

例如 git commit -m "首次上传"

# 第五步：切换到 main 分支

执行命令 git branch -M main

# 第五步：将本地仓库的代码和远程仓库作关联

执行命令 git remote add origin https://github.com/*****/****.git

# 第六步：将本地仓库的代码推送到远程仓库即可

执行命令 git push -u origin main

执行完以上命令，我们就将本地的代码提交到了远程 GitHub 仓库。

# 踩坑

这是因为没有用第五步命令切换到 main 分支，目前 GitHub 新建库默认分支为 main 而不是 Git 默认的 master 。
