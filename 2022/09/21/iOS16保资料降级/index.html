iOS16保资料降级

升级了 iOS16 新系统各种问题，耗电严重，软件不适配，后悔了想降级怎么办？趁官方还没有关闭验证通道，可以尽快降级。
目前 iOS 15.6.1 于 2022 年 9 月 20 日凌晨已经关闭了验证通道，当前 iOS16.0 的用户还可以降级回到 iOS15.7 或者 iOS15.6rc 版本。刷机有风险，刷机需谨慎。
很多人反馈通过爱思保留资料降级，会出现白苹果的情况，另外通过爱思助手备份全资料，再通过全恢复也是不行的，因为在全恢复数据时，高 iOS 版本的全备份文件不能恢复到低 iOS 版本的移动设备内。那怎么办呢，难道就没有完美的保资料方式了么？
当然有的，为了避免出现这种情况，题主我已经帮你们踩坑了，通过我一早上的实操有以下这种方法可以帮你保资料降级。

题主备份设备：MacBook 电脑，iPhone 13PM

备注：Windows 电脑操作流程基本一致

# 1. 备份

首先降级之前，先通过 iTunes 备份你当前的资料。iTunes 里面是点击 “立即备份”。
备份资料完成以后，找到这个备份文件：

MacOS 系统的 iTunes 备份文件默认在：资源库 / Application support/Mobilesync/Backup。
MacOS 系统如找不到这个文件，点击访达 Finder，点击左上角前往，按住 option 键就可以找到资源库，依次进入这个文档就可以找到
win7 或以上的系统会在：C:/Users/ 用户名 / AppData/Roaming/Apple Computer/MobileSync/Backup。

用户名一般是：Administrator，具体根据你电脑名字来。

找到你的备份文件，再找到 Info.plist 这个文件，如图：

MacOS 系统可以通过 Visual Studio Code 打开这个文件 Info.plist，Windows 可以通过记事本的方式打开（博主注:windows 下配置文件最好还是用 Visual Studio Code 打开）；
打开以后，MacOS 系统同时按 Command+F，Windows 系统同时按 Ctrl+F，搜索你刚才备份系统的版本号，比如你备份时系统为：iOS 16.0，则搜索 16.0，iOS 15.7 则搜索 15.7（在 Product Version 的下方可以找到如图版本号），然后将该数字改为低版本号的数字如：16.0>15.5，或 15.7>15.5。原理将你当前备份的版本强行改为更低版本的备份资料，因为低版本的备份可以用于高版本的系统恢复。

  • 低版本恢复高版本，可行
  • 高版本恢复低版本，不可行。

修改完成以后，关闭保存这个 Info.plist 文件。

刷机前，你可以在设置 > 账号 > 查找 > 查找我的 iPhone > 关闭，这样后面就不要你输入密码锁和激活锁了。
下面是进行常规刷机的环节：如 iOS 16.0>15.7 可以通过爱思助手下载

下载完成后，就可以进行刷机，千万不要勾选保留用户资料刷机，这样很容易白苹果。且前面做的事情就白费了，题主是 MacOS 系统，如果是 Windows 系统，爱思助手则勾选用常规快速刷机模式。

静静等待刷机成功就可以了，过程中千万不要拔插设备。

刷机完成后，进入 iPhone 激活页面，记得要将 iPhone 设置为新设备，我们会通过 iTunes 来恢复备份。

那如果想降级到 iOS15.6 的小伙伴，还有其他方法么？某吧很多小伙伴反馈，iPhone 13 系列中 iOS15.6 正式版续航好，玩游戏不掉帧，堪称 iOS15 系列中体验最好的版本，不过题主自己认为，适合自己最重要，哪个版本流畅、续航 OK 就保留哪个版本，不用太纠结这个版本。

但如果有小伙伴喜欢折腾想降级回正式版 iOS15.6 话，目前还可以通过降级到 iOS15.6rc，再通过延迟升级来升级到当前的 iOS15.6 版本，具体教程我放在最下面，各位小伙伴自己斟酌。

通过以下链接下载 iOS15.6rc 版本，题主我是 iPhone 13PM，其他小伙伴自己的型号来下载对应的 iOS15.6rc 版本。
下载完成以后，你可以通过爱思助手选择本地固件来刷机。
没有爱思助手的，也可以通过 iTunes 来。MacOS 系统同时按住 “Option + 检查更新”，Windows 系统同时按下 “Shift + 检查更新”，选取将要刷入的固件等待刷机完毕。
等待刷机完成，iPhone 到了激活界面之后，我们再打开 iTunes，点击恢复备份，选择刚刚已经修改过的备份文件，等待备份完成即可。

经过题主一早上的测试，使用这个方法恢复备份后所有的资料都可以完整恢复，包括照片、通讯录、微信 / QQ 聊天记录等，甚至连微信号都不需要重新登录，直接打开就能与好友聊天了。
