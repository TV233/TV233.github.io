e5移动端office 365出现禁止在你的设备上编辑解决探索

首页

今天打开 iPad 上的 ppt（e5 账户登录），显示 “此帐户禁止在你的设备上进行编辑，对于具有完全访问权限的帐户，请联系你的组织咨询订阅计划。” 以下为个人探索的解决方案。

# 打开仪表盘

在 https://www.microsoft365.com/appsMicrosoft 左边栏管理进入 Microsoft 365 admin center。

# 查看许可证

左边栏 - 用户 - 活跃用户 - 点击你的用户 - 许可证与应用 - 应用，展开翻到下面，发现

这个是灰色的无法选中。

# 开启方法

访问 https://admin.microsoft.com/EAdmin/Device/IntuneInventory.aspx

点击 “我们开始吧”，然后等一下。

正常的话一会后页面自动刷新，只剩一个管理设备的按钮。

再回去看许可证，可以发现已经勾上了。

再在 iPad 上的 ppt 里点一下登录即可正常编辑力
